Just Launch Doom

Super basic Doom launcher

  • Select your Doom executable (e.g. gzdoom)
  • Select IWAD (e.g. doom.wad, doom2.wad)
  • Select PWAD folder (folder where the WADs PK3s are)
  • Select PWAD(s) to run
  • Just Launch Doom!
